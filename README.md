# battleship-game
Battleship Game

A classic Battleship game implemented as a web application using HTML, CSS, and JavaScript.
How to Run

    Clone this repository to your local machine
    Make sure you have the required media files (board.jpg, ship.png, miss.png) in a media folder
    Open the index.html file in your web browser

How to Play

    Enter coordinates in the input field (e.g., "A0", "B3", "F6")
    Click "Fire!" or press Enter to make your guess
    The game will display whether you hit or missed a ship
    Try to sink all 3 ships in as few guesses as possible

Game Board

The game is played on a 7x7 grid with coordinates:

    Letters A-G for columns
    Numbers 0-6 for rows

Technologies Used

    HTML5
    CSS3
    Vanilla JavaScript

Features

    Visual feedback for hits and misses
    Guess counter
    Ship sinking detection
    Keyboard support (Enter key to fire)
